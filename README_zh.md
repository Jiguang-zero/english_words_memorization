# 单词背诵软件(高度定制化)
虽然这个想法在很久之前就已经诞生了，但是由于各种原因(懒癌)，导致一直搁浅。

2024/4/27，在遭到一番神的质问之后，开始动手写。

> 在个人软件开发中，我觉得有一点相当的重要，就是前期不要过度优化，这样很容易消耗热情。

## 项目预期
> 关于此软件的设想，是一个基于命令的定制化单词背诵软件

关于命令，大致有以下几种:  
`insert` 插入单词模式，录入单词模式  
`select` 选择  
`recite` 背诵模式  
`help` 帮助  
`setting` 设置  
各命令可以复用，预想如下: `recite | select -date 2024-1-2, 2024-1-3, between 2024-5-3 2024-5-4`

## 初期目标
+ 添加单词
+ 复习单词
+ 随机默写单词
